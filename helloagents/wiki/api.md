# API 手册

## 概述

当前工程以 **前端演示数据源** 为主，`src/api/*` 提供服务层抽象，便于后续接入真实后端 API。

---

## 认证方式（演示）

- 通过 `auth` store 维护 token/登录态
- 路由守卫根据登录态决定是否允许访问受保护页面

> 注意：当前未实现生产级权限鉴权与 token 刷新策略。

---

## 接口列表（内部服务层）

### Alerts

#### 获取告警列表（演示）
**文件:** `src/api/alertService.js`  
**描述:** 返回告警历史/当前告警（演示数据源）。

#### 导出告警
**文件:** `src/api/alertService.js`  
**描述:** 导出告警数据为 CSV/文本（使用 `src/utils/download.js` 触发下载）。
