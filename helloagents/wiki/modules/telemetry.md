# Telemetry

## 目的
提供系统资源与进程等演示遥测数据的管理与展示入口。

## 模块概述
- **职责:** CPU/内存/磁盘/网络/温度/进程等演示数据管理
- **状态:** ✅稳定
- **最后更新:** 2025-12-25

## 规范

### 需求: 周期性刷新遥测数据
**模块:** Telemetry  
以合理刷新周期更新展示数据，保持 UI 动态感且避免过度刷新。

#### 场景: 自动刷新
- 预期结果：进入主界面后自动刷新；退出/离开主界面后停止定时器（避免资源泄露）
- 说明：刷新由 `src/stores/runtime.js` 统一编排，避免各组件重复创建多个 interval

## 依赖
- `src/stores/telemetry.js`
- `src/stores/runtime.js`
- `src/components/charts/*`
