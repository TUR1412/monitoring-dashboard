# Quantum Ops Monitoring Dashboard

> 量子级监控中枢：实时态势、风险预警与性能洞察的一体化可视化平台。

## ✨ 亮点
- **全域态势总览**：Bento Grid 信息结构，关键指标一眼聚合。
- **沉浸式视觉系统**：极光渐变 + 玻璃拟态 + 细节噪点纹理，拒绝死白与死黑。
- **一体化导航体验**：顶部状态、标签联动与侧边栏分区统一。
- **可持续扩展**：模块化路由与组件结构，便于追加业务域与图表维度。

## 🧭 功能模块
- 仪表盘总览
- 系统资源（CPU / 内存 / 磁盘 / 网络 / 温度）
- 系统警报（当前 / 历史 / 设置）
- 安全中心（威胁检测 / 访问控制 / 合规审计）
- 数据分析（流量 / 性能 / 报告）
- 用户体验与用户管理

## 🧱 技术栈
- Vue 3 + Vite
- Pinia / Vue Router
- Chart.js / Element Plus
- Tailwind CSS + 自定义设计系统

## 🧩 设计系统摘要
- **主色**：Aurora Teal / Solar Amber
- **背景**：多层渐变 + 噪点纹理
- **组件**：玻璃拟态卡片、圆角胶囊按钮、柔和阴影
- **动效**：渐变流动、轻量浮动、聚焦态高亮

## 📁 目录结构（摘要）
`
src/
  components/   # 业务组件与图表
  layouts/      # 页面布局
  stores/       # Pinia 状态
  views/        # 路由页面
  utils/        # 通用工具
`
